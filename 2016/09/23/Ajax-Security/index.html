Ajax

Ajax是一项技术，javascript是一门语言，两者有着本质的区别。Ajax，全称为Asynchronous Javascript And XML,即异步的Javascript和XML，这里有三个点：异步、Javascript、XML。
异步：不阻塞，客户端发送完请求，不用等待服务器响应，直接进行其他操作
Javascript：Ajax采用的Javascript语言，由Javascript构成
XML：指的是数据传输格式，现在多用json

Ajax的同源策略

Ajax是严格遵守同源策略的，既不能从另外的一个域读取数据，也不能发送数据到另外一个域。不过，有一种情况，可以发送数据到另一个域，W3C的新标准中，CORS(Cross-Origin Resource Sharing)开始推进浏览器支持这样的跨域方案，现在的浏览器都支持这样的方案了。
CORS详细讲解见阮一峰的网络日志：http://www.ruanyifeng.com/blog/2016/04/cors.html
有了CORS，跨域从此变得方便了。
