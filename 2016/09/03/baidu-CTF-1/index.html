Code Web

看到题目，感觉这题应该会code，果然，不但要code，还要先看懂code
打开题目，看到一个萌妹子，挺好看的，查看源代码，发现图片并不是通过url加载的，而是<img src='data:image/gif;base64,xxxxx>，google发现如下都是可以的：

我们可以发现url中有个jpg参数，改之为jpg=index.php得到index.php的源码，

这题的难点就在于找flag文件，这里要知道：phpstorm的源码泄漏问题：
PhpStorm是一个php 的IDE。由于编辑器为了防止突然断电，保存项目。都会建立自己的文件夹。
每个项目的配置存储在项目所在目录的 .idea 文件夹中，并以XML格式保存配置。如果你设置的是 “default project settings 默认项目设置”，那么这个默认设置将会自动应用到下一个最新创建的项目上。
于是就找.idea的文件夹，找到了这个http://218.76.35.75:20106/.idea/workspace.xml,从中找到文件fl3g_ichuqiu.php。
因为index.php的代码会过滤参数jgp的值，把下划线删除，把config替换为_,所以，应该访问jpg=fl3gconfigichuqiu.php.
